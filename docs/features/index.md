# Features

SystemSweep combines five system management tools into a single native-looking app.

## At a Glance

- **Dashboard** — Unified health score, resource gauges, quick process view
- **Startup Items** — Scan `~/Library/LaunchAgents`, `/Library/LaunchAgents`, and `/Library/LaunchDaemons`; filter by scope, vendor, or status; bulk disable
- **Browser Extensions** — Parse Chrome `manifest.json` and Firefox `extensions.json`; sort by size; flag high-permission extensions
- **Background Processes** — Live `ps aux` snapshot; CPU/memory bars; kill user processes
- **Disk Analyzer** — Recursive home directory scan with interactive sunburst chart; click-to-drill-down; inspector panel

All scanning happens server-side at render time — no persistent daemons, no background agents. Refresh the page to get fresh data.
